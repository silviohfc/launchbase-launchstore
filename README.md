![PrintScreen](#)


## Instalação
Clone o repositório:
``` 
$ git clone https://github.com/silviohfc/launchbase-launchstore
```
Dentro da pasta instale as dependências:
``` 
$ npm install
```
Inicie o projeto:
``` 
$ npm start
```
